Hamza Mohrach'sProgram — 10 Months

2 sessions per week • 2 groups (same plan) • PPG → PPS → PPI → Tactical Integration
Legend
Physical (PPG) Technical (PPS) Tactical / Positional Game / Scrimmage Recovery / Mobility

Month 1 — PPG (Physical Preparation)

Goal: build base — speed endurance, strength, agility, mobility, plyometrics, resistance; integrate ball late.
Week 1 — Speed Endurance & Strength (Circuit)
Session 1 Speed‑endurance shuttles • Timed runs • Core circuit • Push/Pull bodyweight • Breathing downreg.
Session 2 Strength circuit: goblet squats • RDLs • Rows • Push‑ups • Planks • Farmer carry (rounds).
Week 2 — Agility, Mobility & Plyo (Circuit)
Session 1 Agility ladders • COD (T‑drill) • Hurdle skips • Hip/ankle mobility flow.
Session 2 Plyo circuit: pogo jumps • box step‑offs • lateral bounds • landing mechanics • mobility cool‑down.
Week 3 — Weight Circuit Integration
Session 1 Full‑body weights (circuit): back squat • bench • pull‑down • split squat • anti‑rotation.
Session 2 Power focus: hex‑bar pulls • push press (light) • medicine‑ball throws • jumps (low contacts).
Week 4 — Resistance + Ball Integration
Session 1 Bands & sleds • resisted sprints • lateral slides with bands • mobility reset.
Session 2 Conditioning with ball: dribble shuttles • finishing under fatigue • free‑throw breathing.

Month 2 — PPS (Technical) — Phases 1–2

Focus: Dribbling fundamentals → Dribble + Shoot + Finish.
Weeks 1–2 — Dribbling
Session 1 Stationary & dynamic dribble • weak‑hand • protection • change‑of‑pace.
Session 2 Pressure dribble vs cones/defender • retreat, cross, spin, behind‑back sequences.
Weeks 3–4 — Dribble + Shooting + Finishing
Session 1 Shooting form • off‑catch • footwork (1‑2, hop) • micans/extended layups.
Session 2 Off‑dribble pull‑ups • stops (jump/stride) • finishing vs pad • FT routine.

Month 3 — PPS (Technical) — Phases 3–4

Focus: Dribble + Pass + Shoot → Small‑number advantages.
Weeks 5–6 — Dribble + Pass + Shoot
Session 1 Live passing windows • give‑and‑go • kick‑out after paint touch • re‑spacing.
Session 2 Drive‑kick‑extra • relocation shooting • skip reads • decision games.
Weeks 7–8 — 1v2, 2v2, 3v2
Session 1 1v2 advantage creation • protection finishes • foul‑to‑finish scenarios.
Session 2 2v2 spacing rules • 3v2 decision‑making • quick hit drills.

Month 4 — PPS (Technical) — Phases 5–6

Focus: 3v3, 4v4, 5v5 → Full‑court matches.
Weeks 9–10 — 3v3, 4v4, 5v5
Session 1 3v3 constraints (no dribble, 1‑dribble, touch paint to score).
Session 2 4v4/5v5 emphasis: spacing, ball/player movement, shot quality.
Weeks 11–12 — Full‑Court Matches
Session 1 Full‑court scrimmage • scoreboard • game roles • subs.
Session 2 Game review • film • free throws under pressure • special situations.

Month 5 — PPI (Physical + Technical Integration)

Blend of PPG qualities with technical situations and transitions.
Week 1 — Plyo + Finishing
Session 1 Plyometrics (low‑mod) • approach jumps • landing control.
Session 2 Finishing package: euros, reverses, floaters • contact finishes.
Week 2 — Passing + Agility/Mobility + Sprints + Finishing
Session 1 Passing on the move • wall/partner series • target windows.
Session 2 Agility & mobility circuit • short sprints • finish under fatigue.
Week 3 — 1v1 Transition
Session 1 1v1 full‑court chase • cut‑off, re‑attack • finishing reads.
Session 2 1v1 outlets • hit‑ahead • decision to attack/slow.
Week 4 — 3v3 & 5v5 Transition
Session 1 3v3 transition rules • wide lanes • early drag screens.
Session 2 5v5 primary/secondary break • time/score situations.

Month 6 — Tactical Integration — Phases 1–2

Posts definitions (attack & defense) • P&R • Free throws → then attack & defense strategies.
Weeks 1–2 — Posts, P&R, Free Throws
Session 1 Roles by post (1–5) on offense/defense • spacing landmarks.
Session 2 Pick‑and‑Roll coverages & reads • FT under fatigue.
Weeks 3–4 — Attack & Defense Strategies
Session 1 Offensive packages: horns, 5‑out, motion rules.
Session 2 Defensive packages: ICE/Drop/Switch • shell drill with tags & rebounding.

Month 7 — Tactical Integration — Phases 3–4

Expanded tactical strategies + friendly games.
Weeks 5–6 — Tactical Strategies + Friendly Games
Session 1 Special situations: ATO, SLOB/BLOB, press break.
Session 2 Friendly game • apply week themes • film notes.
Weeks 7–8 — Tactical Strategies + Friendly Games
Session 1 Scout‑style adjustments • matchup plans.
Session 2 Friendly game • special focus (e.g., rebounding margin, TOs).

Month 8 — Tactical Integration — Phases 5–6

Tactical consolidation + friendly games to reinforce identity.
Weeks 9–10 — Tactical Strategies + Friendly Games
Session 1 Late‑game packages • two‑for‑one • foul/defend up 3.
Session 2 Friendly game • emphasize end‑game execution.
Weeks 11–12 — Tactical Strategies + Friendly Games
Session 1 Defensive toggles: switch/hedge/trap rules by lineup.
Session 2 Friendly game • post‑game FT + recovery.
